# ScreenCleanModeWiki
Screen Clean Mode helps you clean your display without accidental clicks, drags, or scrolling.
Launch the app to enter a full-screen cleaning mode that keeps your screen visible while minimizing unintended input.

Highlights
	•	One-tap full-screen cleaning mode
	•	Minimizes accidental clicks, drags, and scrolling
	•	Easy exit: press and hold the left mouse button for 5 seconds
	•	Countdown indicator while holding, so you always know what’s left
	•	Lightweight and fast

Privacy
	•	No personal data collected
	•	No data uploaded
	•	Everything happens locally on your Mac
